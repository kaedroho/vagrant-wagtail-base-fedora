vagrant-wagtail-base (Fedora version)
=====================================

A Vagrant box based on Fedora 22, with the dependencies for developing Wagtail
sites preinstalled.


Build instructions
------------------
To generate the .box file:

    ./build.sh

To install locally:

    vagrant box add wagtail-base-v1.1.1 wagtail-base-v1.1.1.box
